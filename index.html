SING PAY

à Propos

Integrateur

SingPay est un portail de paiement électronique via mobile money: Les services cloud de SingPay communiquent avec les systèmes d’informations des moyens de paiement électronique fournis par les partenaires de SingPay.

Passerelle

SingPay fourni une API RESTfull vous permettant d’effectuer les requêtes de mobile money qui s’effectues nativement dans le contexte GSM depuis une solution fonctionnant dans le contexte du web. SingPay assure la fiabilité, la securité et la régulation de ces translations de domaine.
