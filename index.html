A Propos

Integrateur

SingPay est un portail de paiement électronique via mobile Money: Les services cloud de SingPay comminiquent avec les systeme d’informations des moyens de paiement electronique fournis par les partenaires de SingPay.

Passerelle

SingPay fourni une API RESTful vous permettant d’effectuer les requêtes de mobile money qui s’effectue nativement dans le contexte GSM depuis une solution fonctionnant dans le contexte du web. SingPay assure la fiabilité, la securité et la régulation de ces translation de domaine.
